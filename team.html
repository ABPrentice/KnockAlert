KEVIN GUNDLACH

BLUETOOTH / FRONT-END

Hi! I handled the Bluetooth receiving side of the app. The Bluetooth component of the front-end was developed using Web-Bluetooth, in combination with a custom C++ node plugin. I wanted to work on this project to work with an awesome group of people, and learn new skills.

CONTACT: Linkedin GitHub

NICOLAS JONES

EMBEDDED SYSTEM

I created the Embedded System with Alton, using Nordic, to relay accelerometer data to Electron. The Embedded System is composed of an accelerometer and a Bluetooth Shield. I worked on this project to get experience with interfacing embedded systems with modern apps.

CONTACT: Linkedin GitHub

BEN HULL

FRONT- END

I created the Electron-React-webscript boilerplate, and created the main functionality of the front-end. I wanted to learn Electron in order to get more experience with frameworks and modern-app development. In addition, this was a great opportunity to learn how different technologies can work together.

CONTACT: Linkedin GitHub

ALTON PRENTICE

ALGORITHM / HARDWARE

I worked on sending data from the embedded system to the desktop application, both in the form on a ping and raw accelerometer data. I also worked on the algorithms that detect door knocks. I wanted to work on this project to get experience with a wide range of tools. I also made the physical box!
